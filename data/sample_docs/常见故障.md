# 常见故障判定与处理

本文档用于帮助客服判断商品故障是否属于「质量问题」，从而决定是否走 **15 天质量问题退换货** 流程。

## 1. 耳机 / 音频类

### 常见故障
- 单边没声音 / 声音断续
- 充电指示灯不亮、无法开机
- 连接蓝牙后频繁断连
- 麦克风没声音

### 判定原则
- 排除使用环境问题（如距离过远、连接其他设备）后仍故障 → **判定质量问题**
- 外观完好、无人为损坏 → 走 15 天质量退换货
- 有明显摔痕、进水痕迹 → **不属于质量问题**，引导用户走付费维修

## 2. 屏幕 / 显示类

### 常见故障
- 屏幕出现亮线、暗斑、坏点
- 触控不灵敏、部分区域失灵
- 显示偏色、闪屏

### 判定原则
- 屏幕坏点 ≥ 3 个或位于显示中心区域 → 质量问题
- 边缘 1 个坏点：在国标允许范围内，不属于质量问题
- 明显摔痕或外屏破裂 → 人为损坏

## 3. 电池 / 续航类

### 常见故障
- 续航明显低于宣传值（< 50%）
- 充不进电、充电变慢
- 电池鼓包

### 判定原则
- 电池鼓包属于安全问题，**立刻判定质量问题** 并优先处理
- 续航问题需用户提供使用环境信息（温度、应用场景）后再判定

## 4. 软件 / 系统类

### 常见故障
- 频繁死机、自动重启
- 升级后无法开机（变砖）
- 部分功能无法使用

### 判定原则
- 升级 OTA 变砖：质量问题，优先换新
- 安装第三方软件 / Root / 越狱后出现问题：**不属于质量问题**

## 5. 处理建议

判定为质量问题后，按 [退换货政策](退换货政策.md) 第 2 条流程办理，运费由我方承担。
若有疑问或边界情况，建议转人工客服判定。
